# 开发

* 安装 nodejs
  * http://nodejs.cn/download/
* clone 代码
  * git clone https://github.com/cikirt/destiny.git
* cd src/destiny-web

* npm install
  > 如果要使用淘宝镜像 npm --registry https://registry.npm.taobao.org install  
  > 如果要使用代理 npm config set proxy http://127.0.0.1:1080  
* npm run dev
  > http://localhost:9527/
